# 🚀 Push NationChain to GitHub

## Step 1: Create GitHub Repository

1. Go to https://github.com/new
2. Repository name: `nationchain`
3. Description: `🌍 Blockchain-powered geopolitical strategy game`
4. Choose: **Public** (or Private if you prefer)
5. **DO NOT** initialize with README, .gitignore, or license (we already have them)
6. Click **"Create repository"**

## Step 2: Push to GitHub

After creating the repository, run these commands:

```bash
# Add GitHub remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/nationchain.git

# Push to GitHub
git branch -M main
git push -u origin main
```

## Step 3: Verify

Visit your repository at:
```
https://github.com/YOUR_USERNAME/nationchain
```

You should see:
- ✅ README.md with full documentation
- ✅ All source code
- ✅ LICENSE file
- ✅ .gitignore configured

## Step 4: Add Repository Topics (Optional)

On your GitHub repository page:
1. Click "⚙️ Settings"
2. Scroll to "Topics"
3. Add these topics:
   - `blockchain`
   - `ethereum`
   - `web3`
   - `nft`
   - `game`
   - `strategy-game`
   - `nextjs`
   - `typescript`
   - `solidity`
   - `hardhat`

## Step 5: Enable GitHub Pages (Optional)

To host documentation:
1. Go to Settings → Pages
2. Source: Deploy from branch
3. Branch: `main` → `/docs`
4. Save
